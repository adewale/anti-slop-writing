# Failure: Decorative contrast

## Original

```txt
The point is not the pelicans. The point is the process.
```

## Why it failed

The rhythm implies a relation before the prose explains it. The sentence may sound decisive, but it does not say how the pelican example carries the process claim.

## Mechanism

Decorative antithesis supplies closure without evidence.

## Better rewrite

```txt
The pelican is useful because it gives the process a small, inspectable carrier.
```

## Rule added or changed

Classify short contrast as earned, compressed, or decorative. Replace rhythm with relation when the relation is still hidden.
